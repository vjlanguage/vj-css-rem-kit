# Changelog

## 0.1.0

- Established `typography-universal.css` as the canonical specification source.
- Added normalized TypeScript spec models for root tiers and token groups.
- Implemented runtime auto-rem helpers and `rem` / `vw` / `vh` utilities.
- Added CSS, SCSS, and LESS generators plus checked-in generated style artifacts.
- Added thin React and Vue adapters for lifecycle integration.
- Added package build scripts, exports, and publish-ready `dist/` output.
- Added initial automated tests for the built package surface.
